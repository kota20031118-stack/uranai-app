# 占いアプリ

## ■ 概要
Javaを使用して制作した簡単な占いアプリです。
誕生月・性別・好きな色を選択してボタンを押すと占い結果が表示されるアプリです。

## ■ 制作目的
①JavaのGUI操作に慣れるため
②switch分を使った処理をより理解するため

## ■ 使用技術
・Java
・Swing（GUI作成）

## ■ 機能
・占い結果のランダム表示
・背景画像表示
・ボタン操作による画面切り替え

## ■ 工夫した点
・背景に画像を表示して占い感をだした
・入力を選択式にして簡単にした
・リセット機能を追加した

## ■ 今後の改善案
・占い種類の追加
・結果保存機能
・デザインの向上
・プログラムコンパクトにしたい
