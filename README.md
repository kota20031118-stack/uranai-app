# 占いアプリ

## ■ 概要
Javaを使用して制作した簡単な占いアプリです。
ボタンを押すとランダムで占い結果が表示されます。

## ■ 制作目的
Java学習のアウトプットとして、
GUIアプリケーションの作成を目的に制作しました。

## ■ 使用技術
・Java
・Swing（GUI作成）

## ■ 機能
・占い結果のランダム表示
・背景画像表示
・ボタン操作による画面切り替え

## ■ 工夫した点
・見やすい画面レイアウトを意識
・直感的に操作できるUI設計
・クラス分割による保守性向上

## ■ 今後の改善案
・占い種類の追加
・結果保存機能
・デザインの向上
